Coding challenge for Affinitas job application

Installation
```
  npm install
```

Test
```
  //backend
  npm run test-back

  //frontend
  npm run test-front
```

Build
```
  npm run build
```

Run
```
  //dev
  npm start

  //prod, requires a build and the address that the app is running on, like so :
  REACT_APP_WS_ADDRESS=ws://<address>:1337/ npm run exec-build
```
